Hand Blaster

Apunta con tu mano y dispara

Usa el cursor (o un toque) para apuntar y dispara a los orbes que aparecen. Es un juego creado para funcionar como una página estática, listo para alojarse en todmagichere.github.com/handblaster o cualquier instancia de GitHub Pages.

  • El reto dura 45 segundos.
  • Cada acierto aumenta tu racha, lo que hace crecer el objetivo.
  • Fallas rompen la racha y encogen el objetivo.
  • Juega con clicks o toques: solo necesitas tu mano.
