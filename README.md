# Hand Blaster

Juego web sencillo donde apuntas y disparas con tu mano (clicks o toques) a orbes que aparecen en pantalla. Pensado para publicarse como página estática en GitHub Pages, por ejemplo en `todmagichere.github.com/handblaster`.

## Cómo jugar
1. Abre `handblaster/index.html` en el navegador (sirve en local o en GitHub Pages).
2. Pulsa **Comenzar** para iniciar la cuenta regresiva de 45 segundos.
3. Toca o haz click sobre los orbes brillantes para sumar puntos y aumentar tu racha.
4. Los fallos rompen la racha y reducen el tamaño del objetivo.

## Publicación en GitHub Pages
- Sube este repositorio a GitHub.
- Activa GitHub Pages apuntando a la rama principal (carpeta raíz).
- La ruta resultante será `https://<tu_usuario>.github.io/handblaster/` y cargará el juego automáticamente.

## Estructura
- `handblaster/index.html`: maqueta principal.
- `handblaster/style.css`: estilos visuales responsivos.
- `handblaster/game.js`: lógica del juego sin dependencias.
